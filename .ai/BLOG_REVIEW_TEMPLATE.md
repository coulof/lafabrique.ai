## Review Checklist

### Front Matter
- [ ] Date format is correct (YYYY-MM-DD)
- [ ] Author exists in .authors.yml
- [ ] Categories are relevant and consistent
- [ ] Title is engaging and descriptive

### Content Quality
- [ ] Introduction clearly states purpose
- [ ] Problem statement is clear
- [ ] Technical content is accurate
- [ ] Explanations are clear and complete
- [ ] Examples are practical and useful
- [ ] Conclusion summarizes key points

### Writing Style
- [ ] Active voice used appropriately
- [ ] Sentences are concise
- [ ] Paragraphs are well-structured
- [ ] Technical terms are explained
- [ ] Tone is professional yet approachable

### Technical Content
- [ ] Code snippets are properly formatted
- [ ] Commands are testable
- [ ] Version information is included
- [ ] Error handling is mentioned where relevant
- [ ] Links to official documentation provided

### Formatting
- [ ] Headers follow proper hierarchy
- [ ] `<!-- more -->` tag is present
- [ ] Images have alt text
- [ ] Lists and bullet points used effectively
- [ ] Emojis/icons used consistently

### SEO & Readability
- [ ] Title is 60 chars or less
- [ ] Keywords used naturally
- [ ] Headings are descriptive
- [ ] Content is scannable
- [ ] Visual elements break up text
